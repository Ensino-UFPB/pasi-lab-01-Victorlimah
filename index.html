Alunos de SI desenvolvem aplicação para organização de salas no Campus IV


Como todo ínicio de período, as dificuldades por parte das coordenações de curso para alocar corretamente as turmas nas salas disponíveis são muitas. Pensando nisso, quatro alunos de SI desenvolveram um sistema para facilitar a alocação das salas, trazendo também algumas características de infraestrutura, como por exemplo, se a sala possui equipamentos como ventilador, ar condicionado, projetor e para quais cursos essa sala está alocada.

O SIGMAS ( Sistema de Gerenciamento, Mapeamento e Alocação de Salas) foi desenvolvido durante a disciplina de Engenharia de Software Avançada pelos alunos: Carlos Gabriel, Matheus Soares, Victor Gabriel e Vinicius Teixeira e já disponível para ser acessada através de conexão à rede do campus (CCAE ou LABS-DCX) e o domínio de acesso é sigmas.ccae.ufpb.br
